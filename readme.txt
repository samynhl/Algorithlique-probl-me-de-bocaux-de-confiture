Description des fichiers sources :
	algorithms.py : contient les programmes correspondant aux trois algorithmes
		  I, II et III.
	main.py : permet de lire un fichier en entrée contenant les données de S, K, V
		  le chemin vers ce fichier doit être indiqué comme argument pour la fonction
		  main(path)

	complexity.py: contient les deux fonctions plot_complexity et varier, pour exécuter
		  le fichier il suffit de choisir smax, kmax et le numéro de l'algorithme

	testGlouton.py : contient les fonctions TestGloutonCompatible, GenererCapacite et la fonction 
		  analyse
	stats.py : contient des fonctions dont le rôle est de donner des statistiques 
		  sur les systèmes de capacités non glouton-compatibles
